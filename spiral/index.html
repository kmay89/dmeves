The Spiral Periodic Table

All 118 elements as what they actually are — one continuous sequence — wound into a spiral where every turn is a period, every spoke is a chemical family, and every number is real.
Why this exists. Most periodic tables either simplify the data or decorate it. This one was built for high-school chemistry and everything beyond it: a layout that shows why the table repeats, wrapped around reference-grade data — IUPAC atomic weights, NIST energies and spectra, measured radii — with the source of every value one click away. Where a number is exact, it says so. Where it's an approximation or a prediction, it says that too.
Click any element for its full data sheet, then open the ψ Quantum view — real solutions of the Schrödinger equation: orbitals, nodes, spectra, and why the atom bonds. Color by any measured property to watch the periodic trends emerge as gradients. Spiral ⇄ Classic morphs between this layout and the familiar flat table — they're the same object. Where every number comes from

The rule this project is built on: exact where exact, measured where measured, and labeled approximation everywhere else. Blank cells mean "no reliably established value" — a blank is more honest than a guess.
